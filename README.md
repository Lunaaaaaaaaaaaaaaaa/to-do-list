# ✨Todo App

A modern and minimal **Todo Application** built with **React** featuring a futuristic glassmorphism design, animated gradients, and a beautiful dark UI.

## 🌟 Features

* ✅ Add and manage tasks
* ✅ Mark tasks as completed
* ✅ Modern glassmorphism interface
* ✅ Animated gradient background
* ✅ Responsive design
* ✅ Smooth hover and transition effects
* ✅ Custom styled checkboxes
* ✅ Minimal and clean user experience

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/Lunaaaaaaaaaaaaaaaa/to-do-list.git
cd to-do-list
```

### Install dependencies

```bash
npm install
```

### Run the development server

```bash
npm run dev
```

Open:

```text
http://localhost:5173
```
